15. Модули itertools, more-itertools
===================================

В этом разделе рассматриваются два модуля: модуль из стандартной библиотеки itertools
и сторонний модуль more-itertools. Оба модуля предоставляют набор функций для работы
с итераторами.

Оба модуля содержат большое количество функций, поэтому в этом разделе рассматриваются
только некоторые из них.
